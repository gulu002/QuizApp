# 嘉年华刷题 App 需求文档

## 一、项目概述

| 项 | 说明 |
|---|---|
| 项目名称 | 嘉年华刷题（暂定） |
| 题库来源 | 《嘉年华版本.docx》，共 **3110 道** 医学综合题 |
| 题型分布 | 单选题 2018 道 / 多选题 733 道 / 判断题 359 道 |
| 当前状态 | 题目文本完整，**答案与解析缺失**，需后续补充 |
| 目标用户 | 医学生、护理考生、医学从业者 |

---

## 二、功能需求

### 2.1 用户模块

| 功能 | 说明 |
|---|---|
| 注册 / 登录 | 手机号 + 验证码，或微信一键登录 |
| 个人信息 | 头像、昵称、学习天数、累计刷题数 |
| 密码找回 | 手机验证码重置 |
| 账号注销 | 支持注销并清除全部数据 |

### 2.2 题库模块

| 功能 | 说明 |
|---|---|
| 题目浏览 | 按题型（单选/多选/判断）、学科分类筛选 |
| 题目搜索 | 关键词模糊搜索题干内容 |
| 题目详情 | 展示题干、选项、正确答案（作答后）、解析 |
| 学科分类 | 支持按学科标签（内科/外科/妇产/儿科/药理/免疫等）筛选 |
| 题库更新 | 后台追加新题、批量导入答案、增量同步 |

### 2.3 练习模式

| 模式 | 说明 |
|---|---|
| **顺序练习** | 按题号顺序逐题作答 |
| **随机练习** | 随机抽取，可设数量（10 / 20 / 50 / 自定义） |
| **题型专练** | 仅练习某一题型（单选 / 多选 / 判断） |
| **分类专练** | 仅练习某一学科分类的题目 |
| **模拟考试** | 按题型比例组卷（如 100 题限时 90 分钟），交卷打分 |
| **错题重练** | 仅练习错题本中的题目，答对多次后自动移出 |
| **收藏练习** | 仅练习已收藏题目 |
| **每日一练** | 每日推送 10 题，连续打卡有统计激励 |

### 2.4 答题交互

| 功能 | 说明 |
|---|---|
| 选项点击 | 单选点击即选中；多选勾选后确认提交；判断二选一 |
| 即时判题 | 提交后立即显示对错、正确答案、解析 |
| 题目切换 | 左右滑动或按钮切换上/下一题 |
| 答题卡 | 总览当前练习进度，快速跳转任意题 |
| 作答标记 | 支持标记不确定的题目，交卷前可回看 |
| 夜间模式 | 深色主题，保护视力 |

### 2.5 错题本

| 功能 | 说明 |
|---|---|
| 自动收录 | 答错的题自动加入 |
| 错题状态 | 记录错误次数、最近错误时间 |
| 自动移出 | 连续答对 3 次自动移出错题本 |
| 手动管理 | 支持手动移除、批量清空 |

### 2.6 收藏功能

| 功能 | 说明 |
|---|---|
| 收藏 / 取消 | 练习中一键收藏 |
| 收藏夹 | 按时间排序，支持筛选题型、分类 |
| 批量管理 | 支持批量取消收藏 |

### 2.7 统计与进度

| 功能 | 说明 |
|---|---|
| 学习日历 | 日历热力图展示每日刷题数量 |
| 正确率 | 总体正确率、按题型/学科维度的正确率 |
| 进度概览 | 已做 / 未做 / 已掌握（连续答对 3 次） |
| 用时统计 | 单题平均用时、累计学习时长 |
| 模考成绩 | 历史模考记录及正确率趋势 |
| 薄弱分析 | 按学科展示薄弱环节，推荐针对性练习 |

### 2.8 管理后台（Web 端）

| 功能 | 说明 |
|---|---|
| 题目管理 | 增删改查、批量导入/导出 Excel |
| 答案管理 | 批量录入答案和解析（Excel 导入） |
| 分类管理 | 创建/编辑学科分类，为题目打标签 |
| 用户管理 | 查看用户列表、活跃数据 |
| 数据统计 | 题库使用热力图、高频错题 TOP100、用户留存分析 |

## 三、非功能需求

| 项 | 要求 |
|---|---|
| 平台 | Android + iOS App（推荐 Flutter / React Native）+ Web 管理后台 |
| 后端 | Spring Boot / Go，MySQL 数据库 |
| 离线支持 | 本地 SQLite 缓存全量题库，无网络也能刷题 |
| 性能 | 3110 题全量加载 < 2s，练习页面进入 < 500ms |
| 数据同步 | 登录后错题/收藏/进度云端同步，多设备数据一致 |
| 安全 | 接口鉴权（JWT）、敏感数据加密、防刷限流 |
| 兼容 | Android 8.0+ / iOS 14+ |

## 四、题型说明

### 单选题（2018 道）
- 选项：4~5 个（A / B / C / D / E）
- 答案：单个字母（如 `C`）
- 示例：「肾结核最常见的症状是」→ A.肾积水 B.尿频 C.肾区疼痛 D.潮热盗汗 E.贫血

### 多选题（733 道）
- 选项：4~5 个（A / B / C / D / E）
- 答案：多个字母排序拼接（如 `BDE`）
- 示例：「下述抗体再次应答的特点,正确的是」→ 多选

### 判断题（359 道）
- 选项：2 个（A.正确 / B.错误）
- 答案：`A` 或 `B`
- 示例：「健康是指…」→ A.正确 B.错误
